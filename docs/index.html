Welcome to MiniDB Documentation

Installation

Once you have installed MiniDB from its official github page, it is time to install the dependencies:

npm install package.json

How to get started

To get started with MiniDB, simply import it in your project. Make sure that the MiniDB folder is in the exact same directory as your project:

// Import MiniDB
const {MiniDB} = require('MiniDB');

//Create new db instance for `test_table`
const db = new MiniDB('test_table');

Function initialize()

Initialize funtion is used to create the table into the db after the new DB instance has been created. It must be called before doing any operations on the db!


 //Create `test_table` table in the database.
 db.initialize();

Function set()

Set function is really easy to understand, it takes key and value as parameters and then stores them in the database.


 //Create `test` key in database with value `1234` attached to it.
 db.set('test', 1234);

Function get()

Get function is important when it comes to reading what's behind a certain key. This function simply takes key as parameter and returns, if the key exists, value as a promise. In case the key doesn't exist, this function returns undefined.


 //Return whatever is behind key `test`.
 db.get('test').then(resp => console.log(resp));

Output:


1234

Function update()

Function update takes two parameters and, as the name suggests, this function updates the value that is behind a certain key. First parameter is key that you want to have it's value updated and the second parameter is new value that will replace the old one.


 //Update value behind key `test` from `1234` to `12345`
 db.update('test', 12345);

Function remove()

Function remove takes only one parameter which is key and it pretty much deletes that key including it's value forever.


 //Removes key `test` along with it's value `12345` from table `test_table`
 db.remove('test');
